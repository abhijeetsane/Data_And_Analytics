# Data Dashboarding and Visualization

## Marketing Data Monitoring

- **WhataGraph** [https://whatagraph.com/]
  - Whatagraph is a cross-channel reporting tool that helps businesses to track, measure, and analyze marketing performance with clients and teams. Users can automatically pull data from all of their marketing channels, create visual, engaging reports, and share them easily.
  - Several benefits of using it are finding correlations, quick action, identifying new trends and also complex concepts are simplified.
  - It has features and the ability to extract and transform data in real-time, so your interactive graphs will change frequently.
  - One can remove or add some components, compare them with one another over a specific time frame, and isolate outliers.
  - It allows you to curate detailed performance dashboards and monitor otherwise complex data in a fairly simple way. To visualize social media metrics or any other marketing analytics, this is a preferred tool.
  
- **Visme** [https://www.visme.co/]
  - Visme’s graph maker has the power to easily create line graphs, bar graphs, pie charts, scatter plots, histograms and many more chart styles. The data widgets in Visme are perfect for visualizing map data, arrays, tables, percentages, gauges and more.
  - Import live data from Google Sheets and create custom dashboards to track business insights. You can also create social media reports, sales reports and goal projections easily using a wide range of ready-to-go templates.

- **Reeport** [https://www.reeport.io/]
  - Reeport is used for the analysis and distribution platform that allows everyone to manage their activity on a daily basis.
  - It is a ready-to-use and self-service data analytics solution that connects natively and instantly via API to all your digital data sources.
  - For each dataset connected to Reeport you can easily rename, re-calculate, re-aggregate or even favorite your metrics and dimensions. Employees work in a familiar environment where the data is understandable by everyone.
  - To facilitate the work of business teams Reeport makes it easy to reconcile data from different ecosystems. You no longer need to launch specific developments or build complex data schemas.

- **D3** [https://d3js.org/]
  - D3 which stands for Data-Driven Documents is a dynamic, interactive, online data visualizations framework used in a large number of websites.
  - It is an extremely powerful visualization tool to create interactive data visualizations that exploits the modern web standards like SVG, HTML and CSS to create data visualization.
  - Allows you to manipulate the Document Object Model (DOM) based on your data. D3 also gives the flexibility to provide dynamic properties to most of its functions where properties can be specified as functions of data. That means your data can drive your styles and attributes.
  - Permits you to create Animations, Interactions, Analysis, Hierarchies, Networks, Bars, Lines, Areas, Dots, Radial, Annotations and Maps.
  
- **C3** [https://c3js.org/]
  - C3 which stands for Comfortable, Customizable, and Controllable which makes it easy to generate D3-based charts by wrapping the code required to construct the entire chart. Here writing D3 code won't be necessary.
  - Gives some classes to each element when generating, so you can define a custom style by the class and it's possible to extend the structure directly by D3.
  - Provides a variety of APIs and callbacks to access the state of the chart. By using them, you can update the chart even after it's rendered.
  - Helps in making charts, Axis, Data, Grid, Region, Interaction, Legend, Tooltip, Transforms and Style.

- **RAW Graphs** [https://rawgraphs.io/]
  - RAW Graphs is an open source data visualization framework that aims at providing a missing link between spreadsheet applications and vector graphics editors.
  - It works with tabular data as well as with copied-and-pasted texts from other applications. Based on the SVG format, visualizations can be easily edited with vector graphics applications for further refinements or directly embedded into web pages.
  - Assists in creating charts, diagrams and plots.
  - Works on sensitive information, the data injected into RAWGraphs is processed only by the web browser: no server-side operations or storages are performed. No one can see, touch or copy your data.

- **Dundas BI** [https://www.dundas.com/]
